Uncertainty in Heat Maps: Task One

In this study, you will be asked to interpret charts of data and uncertainty, and make predictions based on these charts. These charts represent maps of a region, where each location has an associated value and an uncertainty about that value. You will be presented with three kinds of charts.

Two side by side charts. The map showing the data is on the left, and the map showing uncertainty is on the right.
A single map where each combination of data and uncertainty is given a different, unique color.
A single map where, as we become more uncertain about data, the number of colors for showing the data decreases.

There are two tasks in this study. In this first task, you will be given a chart, and asked to select a region in the chart that meets some criteria. For instance, you might be asked to click on a region with value of 0.2, and an uncertainty of 0.1. Select any location you see that meets the requirements. Click on a location to input your answer.

An example of the first task. Any of the locations highlighted in red would be valid answers.
